# NDVI Outputs

This folder contains packaged outputs for easier review.

- `final_bridge_analysis.csv`: Final catastrophe-model dataset after NDVI damage classification, fragility, and disruption analysis. Currently present.
- `proxy_validation_metrics.csv`: Proxy-validation metrics comparing the HAZUS baseline against the hybrid validation model. Currently present.
- `proxy_validation_predictions.csv`: Held-out bridge-level proxy-validation predictions for HAZUS and the hybrid validation model. Currently present.
- `NDVI_Raster_Maps.png`: Three-panel raster comparison of pre-event NDVI, post-event NDVI, and NDVI change. Currently present.
- `NDVI_Change_Only.png`: Standalone map of NDVI change. Currently present.
- `PGA_vs_NDVI.png`: Scatter plot comparing bridge PGA values to bridge-level NDVI change. Currently present.
- `Bridge_NDVI_Map.png`: Spatial bridge map colored by NDVI change. Currently present.
- `Phase1_DamageClassification.png`: Damage-state distribution figure based on NDVI thresholds. Currently present.
- `Phase2_FragilityCurves.png`: Empirical proxy fragility curves derived from NDVI-based damage states. Currently present.
- `Phase3_EconomicDisruption.png`: Traffic disruption summary figure. Currently present.
- `Phase4_PriorityMap.png`: Emergency bridge prioritization map. Currently present.
- `proxy_validation_confusion_matrices.png`: Side-by-side confusion matrices for the HAZUS baseline and the hybrid proxy-validation model. Currently present.
