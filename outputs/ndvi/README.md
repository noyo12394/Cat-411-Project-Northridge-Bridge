# NDVI Outputs

This folder contains packaged outputs for easier review.

- `final_bridge_analysis.csv`: Final catastrophe-model dataset after NDVI damage classification, fragility, and disruption analysis. Currently not present yet.
- `NDVI_Raster_Maps.png`: Three-panel raster comparison of pre-event NDVI, post-event NDVI, and NDVI change. Currently not present yet.
- `NDVI_Change_Only.png`: Standalone map of NDVI change. Currently not present yet.
- `PGA_vs_NDVI.png`: Scatter plot comparing bridge PGA values to bridge-level NDVI change. Currently not present yet.
- `Bridge_NDVI_Map.png`: Spatial bridge map colored by NDVI change. Currently not present yet.
- `Phase1_DamageClassification.png`: Damage-state distribution figure based on NDVI thresholds. Currently not present yet.
- `Phase2_FragilityCurves.png`: Empirical proxy fragility curves derived from NDVI-based damage states. Currently not present yet.
- `Phase3_EconomicDisruption.png`: Traffic disruption summary figure. Currently not present yet.
- `Phase4_PriorityMap.png`: Emergency bridge prioritization map. Currently not present yet.
