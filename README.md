# Cat-411-Project-Northridge-Bridge

This repository contains bridge-level earthquake exposure analysis for the
1994 Northridge earthquake as part of our CAT 411 course.

## Week 1 – Bridge Exposure
- Processed the California National Bridge Inventory (NBI)
- Converted bridge coordinates to geographic (lat–lon) format
- Integrated USGS ShakeMap Peak Ground Acceleration (PGA) data
- Assigned PGA values to ~16,800 bridges within the ShakeMap footprint
- Generated maps and statistical plots to validate exposure patterns

## Contents
- `Bridge_Week1.ipynb` – Week 1 notebook with data processing, exposure analysis, and visualizations
