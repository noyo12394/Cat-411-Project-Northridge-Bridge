# Data Layout

This project now expects data relative to the repository root.

Included now:
- `CA25.txt`
- USGS ShakeMap rasters such as `pga_mean.flt` and `pga_mean.hdr`
- reference files used during the original project

Generated by the notebooks:
- `processed/pga_nbi_bridge.csv`
- `processed/bridges_with_pga_affected_only.csv`
- `processed/bridges_with_edr.csv`
- `processed/bridges_with_svi.csv`
- `processed/bridge_ml_predictions.csv`
- `processed/final_bridge_analysis.csv`

Optional for `run_analysis.ipynb`:
- `change_detection/Pre_Event_NDVI.tif`
- `change_detection/Post_Event_NDVI.tif`
- `change_detection/NDVI_Change.tif`
- `change_detection/pga_bridge_ndvi_200m.csv`
- `change_detection/pga_bridge_ndvi_200m.shp` plus companion shapefile files

If those optional change-detection files are missing, the notebook will stop with a clear message explaining what needs to be added.
